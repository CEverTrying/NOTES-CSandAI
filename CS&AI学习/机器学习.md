# 分类&回归

- **分类**(classification): 标签 (label) 是离散 (discrete) 值
- **回归**(regression): 标签 (label) 是连续 (continuous) 值

## 训练过程（以垃圾分类为例）
- 训练数据：由输入样本和真实标签（**监督学习**）组成
- 特征提取：如将关键词转化为词频向量，再结合标签，得到结构化训练数据集
- 模型训练：尽可能使得训练的后的标签预测与训练数据集相符
- **关键概念**: 训练误差，测试误差，训练集和测试集，过拟合，欠拟合[^1]

[^1]: **训练集 / 测试集**：训练集用于模型学规律；测试集评估泛化能力，不参与训练。 **训练 / 测试误差**：训练集上的预测误差叫训练误差；测试集上的叫测试误差。 **欠拟合**：模型太简单，训练、测试误差都大，没学透规律。**过拟合**：模型太复杂，训练误差极低但测试误差高，死记训练数据。

## 参数

简单的机器学习算法（同时也是**非参数化模型**）例子：k临近算法：对于一个测试样本，用训练样本中距离它最近的k个样本中占多数的标签 来预测测试样本
- 不需要训练，只需要一个距离函数
- 缺点：需要储存所有训练数据，需要计算大量距离，距离不能代表相似度
- **维度灾难**：在高维中样本的**密度被大幅地稀释**，会带来计算量爆炸、过拟合风险飙升等一系列问题

**非参数化模型与参数化模型相比**：需要保存训练样本，不被有限参数定义/不包含参数

最简单的参数化模型——**线性模型**
- 关于相关公式，梯度下降，局部最小值，参考[[CV导#机器学习基础上的cv|cv导]]（*疑似cv导已经可以解决大部分问题了*）
- 最小化凸函数称为凸优化

## 分类问题
经验风险最小化框架（ERM）：
- 首先确定采用的模型f（如线性模型）
- 其次决定损失函数
- 在训练集上最小化loss function平均值

### 多分类
k分类问题共同训练k个模型（emm），然后用softmax将k个数据转化为概率分布

### 过拟合和正则化
为了解决过拟合问题，引入了正则化

- 实际中一般会在损失函数中加入一项 $\lambda R(f)$其中lambda是一个超参数，不随训练优化
- 这个R是为了应对一些常见的过拟合原因：
	- 可能是某些特征维度支配了预测，即有些权重w_j太大。那么引入L2正则化，R（f）=sum w^2 ,用于惩罚过大的权重
	- 可能是数据中存在大量没用的特征维度，但是赋予了他们非0的权重。那么引入L1正则化，R（f）=sum|w|，鼓励稀疏的w

### 超参数和模型选择
对于选择最优超参数的问题，使用==验证集==（validation set）：对于每一种超参数的组合，在训练集上训练参数，在验证集上验证模型误差，找到在**验证集上误差最小的模型和超参数**，最终在测试集上最终测试模型误差

应对总数据集太小，验证集不能准确验证模型性能的问题：**k-折交叉验证**：
- 将测试集以外的所有数据随机分为k份
- 对于每一种个给定的超参数的组合：每次使用k-1份数据训练模型，用1份验证模型误差。k次测试误差最小的就是选中的超参数组合

（*然而现在大模型数据太大以至于无法做到*）

### 决策树和随机森林

#### 决策树
**定义**：一种**树状结构**，包含一个**根节点**（包含样本全集），若干**内部节点**（对其中的样本集合的特征进行测试后，按结果分配样本到不同字节点）和若干**叶节点**（对应决策结果）（取多数类[^2]）

[^2]: 取多数类指的是在**训练过程**中，如果有多个不同标签的样本被分到了同一个叶节点，那就选择最多的那种标签

理想情况下想要让叶节点只包含同类型的样本（也就是对样本进行完全正确的分类）。这需要我们找到最合适的**划分属性**：
- 希望随着划分的进行，决策树分支节点包含的样本子集的**“纯度”**不断增加。纯度的度量：
	- ==信息熵==
	- ==基尼指数==
- 希望找到一个好的划分属性，根据属性的不同取值可以将当前节点的样本划分，使划分后的子集只包含同类的标签。划分标准的选择：
	- ==信息增益==
	- ==增益率==
	- （属性的）基尼指数

**信息熵公式**：
- 衡量一个事件的信息量：发生概率越低的事件的信息量越大 ，计算方法为$log_2 (\frac{1}{P(x_i)})$
- $H(X)=-\sum_{i=1}^{n} P(x_i)\log_2 P(x_i)$
- 条件信息熵：$H(X\mid Y) = -\sum_{y}\sum_{x} P(x,y)\log_2 P(x\mid y)=\sum_{y} P(y)\,H(X\mid Y=y)$
- 联合熵：$H(X,Y)-\sum_{x}\sum_{y} P(x,y)\log_2 P(x,y)$
- 互信息：$I(X,Y)=H(X)+H(Y)-H(X,Y)$
- 信息增益：$\text{Gain}(X,Y) = H(X) - H(X\mid Y)$
这时就有一个划分的标准为：一个属性A对集合D的**信息增益**。$$g(D,A)=H(D)-\sum_i\frac{|D^{A=a_i}|}{|D|}H(D^{A=a_i})$$（对应上式中Y为标签D，X为属性A）.其中$D^{A=a_i}$为D中属性A为ai的子集

然而有时属性的信息增益高**并不代表具有泛化能力**（如将序号看做划分属性），信息增益准则倾向于**选择具备更多取值可能的属性**。

==增益率==：
- 定义：$$g_R(D,A)=\frac{g(D,A)}{H_A(D)}$$
- 其中H_A（D）描述A本身的熵：$H_A(D)=-\sum_i\frac{|D^{A=a_i}|}{|D|}log_2\frac{|D^{A=a_i}|}{|D|}$
- 这个算法一定成度上缓解了属性本身高信息的增益

==基尼指数==：
- 基尼指数：$Gini(D) = 1 - \sum_{k=1}^{K} p_k^2$ 
- 条件基尼指数： $Gini(D\mid A) = \sum_{i=1}^{n} \frac{|D_i|}{|D|}\,Gini(D_i)$
- 基尼增益： $\Delta Gini = Gini(D) - Gini(D\mid A)$
- 应选择划分后基尼指数最小的属性：min（Gini（D，A））

**树模型训练伪代码**：
```
函数 BuildTree(数据集 D, 特征集 A): 
	// 终止条件1：样本全属于同一类别 
	if D 中所有样本标签相同: 标记为叶节点，返回该类别 
	// 终止条件2：特征集为空 / 样本太少（预剪枝） 
	if A 为空 或 样本数量 < 阈值:
		标记为叶节点，返回D中多数类别 
		
	// 1. 遍历所有特征，选最优划分特征 
	初始化 最优特征 = None 
	初始化 最优指标 = 极小值 
	for 每个特征 a ∈ A: 
		计算 当前划分指标 
		// 熵体系：信息增益 Gain = H(D) - H(D|a) 
		// 基尼体系：基尼减少量 ΔGini = Gini(D) - Gini(D|a) 
		if 当前指标 > 最优指标: 
			最优指标 = 当前指标 
			最优特征 = a 
			
	// 2. 根据最优特征a划分数据集 
	依据最优特征a的取值，将D划分为子集 D₁,D₂,...,Dₙ 
	// 3. 移除已使用特征 
	新特征集 A_new = A - {最优特征} 
	
	// 4. 递归构建子树 
	创建当前内部节点 = 最优特征 
	for 每个子集 Di: 
		子树 = BuildTree(Di, A_new) 
		将子树挂载为当前节点分支 
	
	return 当前节点
```

**属性取值连续**：
- 对于m个取值可以选取m-1个候选的划分点，选取**能最大化信息增益**的划分点
- 在此之后，选取信息增益最大的候选属性

**标签连续**：
- 分类问题到回归问题
- 决策树到回归树

#### 回归树
- 保留决策树的结构
- 通过L2 loss定义集合的纯度:$L(D)=var(D)*|D|    L(D,A)=\sum_iL(D^{A=a_i})$，要求最小化L（D，A）

#### 集成学习&随机森林
- ==集成学习== =多个个体学习器+结合策略
- 如：随机森林
	- 个体学习器：决策树
	- 步骤
		- **样本扰动**：每一个决策树，对训练集随机采样得到一个独立的训练集
		- **属性扰动**：每一个决策树，只选择数据集中的一部分样本特征进行子树划分训练
		- 最后，用所有训练好的决策树的**平均预测（如多数类）**作为对训练样本的输出
